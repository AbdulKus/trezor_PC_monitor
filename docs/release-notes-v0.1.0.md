# Trezor PC Monitor v0.1.0

The first public release packages the Windows editor, PC telemetry service, pixel-accurate media pipeline and dedicated Trezor One firmware.

Highlights:

- six editable default OLED layouts;
- live 1-bit preview for images and animated GIFs;
- final-size resource compilation and 64-KiB package accounting;
- CPU, RAM, GPU/VRAM and PresentMon FPS channels;
- reliable HID reconnect and transactional layout uploads;
- short/long button actions and local page switching;
- portable Windows bundle and both `TRZF`/`TRZR` firmware images.

This is an early unofficial community release. Use a dedicated Trezor One, not a wallet, and read the flashing warning in the repository before installing.
